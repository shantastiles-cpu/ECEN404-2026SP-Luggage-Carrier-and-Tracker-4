# 🧳 Luggage Carrier and Tracker
ECEN 404 - Senior Design 2026

A smart luggage-carrying rover that automatically follows its owner, avoids obstacles, and provides real-time tracking to relieve user fatigue and improve mobility.

## 📌 Project Overview
Problem Statement: Carrying luggage through large areas causes fatigue and logistical issues, particularly for the elderly or disabled.

Solution: An autonomous rover capable of following a designated user while navigating complex environments safely.

## 🛠 Subsystem Breakdown
This project is divided into four major technical subsystems. Detailed documentation and source code for each can be found in their respective directories:

App Subsystem: Flutter-based mobile application for user interface, luggage control, and Supabase integration.

Navigation Subsystem: Core logic for path planning and motor control.

Obstacle Avoidance Subsystem: Sensor integration (LiDAR/Ultrasonic) to ensure safe travel.

Tracking Subsystem: UWB or GPS-based logic for "follow-me" functionality.
